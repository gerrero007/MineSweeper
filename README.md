# Minesweeper

### Primarily for Joan Llavors. (Design of his choice).

A simple Minesweeper game based on the originals, for playing without ads or distractions. Includes time limits and different sizes for practice.

## Settings Menu:

- Changing the theme to three options: original (green/orange), dark (black/blue), and light (white/green).

- Maximum number of records: min 1, max 100

- Enable/Disable timer
